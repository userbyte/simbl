simbl setup

hello! thank you for using simbl :)
simbl is a solo microblogging platform, designed with simplicity in mind.
to get everything setup, you just need to setup the account which will be the primary admin of this simbl instance.

simbl uses another one of my projects, xAPI, for authentication.
if you are self-hosting simbl, you'll need to setup an xAPI instance.

to setup simbl:
- enter the API URL of the simbl-server you're setting up
- enter an xAPI instance URL
- login to an account that exists on that xAPI instance
- return to the homepage and start posting!
